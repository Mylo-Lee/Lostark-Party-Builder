# Lostark Party Builder

로스트아크 4인 파티 조합 구성을 도와주는 웹 어플리케이션입니다. 검색한 캐릭터의 파티 구성 및 정보를 제공하며, 캐싱 기능과 API 통합이 포함되어 있습니다.

## 시작하기

이 프로젝트는 Node.js (Express) 환경에서 실행됩니다.

### 1. 패키지 설치
프로젝트 폴더에서 아래 명령어를 실행하여 필요한 의존성을 설치합니다.
```bash
npm install
```

### 2. 환경변수 설정
프로젝트 최상단 폴더에 `.env` 파일을 생성하고, 발급받은 로스트아크 API 키를 입력합니다.
```env
LOSTARK_API_KEY=당신의_로스트아크_오픈API_키
PORT=3000
```

### 3. 로컬 서버 실행
```bash
npm start
# 또는 개발 모드: npm run dev
```
서버가 실행되면 웹 브라우저에서 `http://localhost:3000` 으로 접속할 수 있습니다.

## 외부 접속 허용 방법 (ngrok 배포)

로컬에서 개발 중인 서버를 외부에 공유하려면 `ngrok`을 사용할 수 있습니다.

1. **로컬 서버를 먼저 실행**합니다 (`npm start`).
2. **새로운 터미널(명령 프롬프트/PowerShell) 창**을 열어줍니다. (기존 서버 실행 창을 닫으면 안 됩니다.)
3. 새 터미널에서 다음 명령어를 실행하여 3000번 포트를 외부에 엽니다.
   ```bash
   ngrok http 3000
   ```
4. 터미널에 나타나는 화면에서 `Forwarding` 항목의 URL(`https://<랜덤문자열>.ngrok-free.app`)을 확인합니다.
5. 해당 주소를 복사하여 다른 사람에게 공유하거나 외부 기기에서 접속하면, 귀하의 로컬 환경에서 구동 중인 웹사이트에 접속할 수 있습니다. (단, 로컬 서버 실행 터미널과 ngrok 실행 터미널이 모두 켜져 있어야 합니다.)

> **주의 ⚠️**
> - 외부에서 접속하는 동안에는 **로컬 서버 실행 터미널**과 **ngrok 실행 터미널**이 모두 켜져 있어야 합니다. 터미널을 닫으면 접속이 끊어집니다.
> - `ngrok` 무료 버전은 실행할 때마다 도메인 주소가 새롭게 변경됩니다.
